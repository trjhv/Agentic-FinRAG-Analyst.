# Agentic-FinRAG-Analyst.
An advanced multi-page app transcending passive RAG. Powered by Llama 3.3 via Groq, it uses a ReAct loop and hybrid search (FAISS+BM25) to fuse static 10-K extraction with live yfinance data. Features deterministic routing,  multi-hop reasoning, and robust free-tier rate-limit resilience for zero-hallucination financial analysis.
